# Tutorial eIDAS

Proyecto introductorio y progresivo sobre el reglamento eIDAS de la Union Europea, pensado como base para documentacion, materiales didacticos y futuras ampliaciones sobre identidad digital, confianza electronica y servicios cualificados.

## Objetivo

Este repositorio nace para construir un tutorial claro sobre:

- que es eIDAS y por que importa en Europa
- como funciona la identidad electronica en el marco europeo
- que son los servicios de confianza
- que diferencia hay entre firma electronica, sello electronico, sello de tiempo y entrega electronica certificada
- que cambios introduce eIDAS 2.0 y la Cartera Europea de Identidad Digital

La idea es que el contenido sea comprensible para perfiles juridicos, tecnicos y de negocio.

## Enfoque del tutorial

El proyecto seguira un formato practico y gradual:

1. contexto general del reglamento
2. conceptos basicos y vocabulario esencial
3. actores implicados
4. servicios de confianza y casos de uso
5. marco de cumplimiento
6. evolucion hacia eIDAS 2.0
