# 16. Bibliografia y fuentes por capitulo

## Introduccion

Este proyecto tiene finalidad didactica y divulgativa. Por ello, resulta util identificar fuentes institucionales y normativas que permitan verificar o ampliar cada uno de los bloques del tutorial.

## Recomendacion de uso

Cuando este proyecto siga creciendo, conviene que cada nuevo capitulo cite al menos:

1. una fuente normativa principal
2. una fuente institucional o de implementacion
3. si procede, una fuente nacional complementaria

## Advertencia metodologica

La existencia de una fuente citada no implica que toda afirmacion del tutorial reproduzca literalmente esa fuente. En varios pasajes se han realizado simplificaciones didacticas para facilitar la comprension general del marco eIDAS.
