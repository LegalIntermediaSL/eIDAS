# 28. Ejemplos documentales y plantillas orientativas

## Introduccion

Esta seccion recoge ejemplos muy simplificados de como una organizacion podria ordenar internamente algunos elementos de su marco de confianza digital. No son modelos juridicos cerrados, sino orientaciones didacticas.

## 1. Politica interna de uso de firma

### Estructura orientativa

1. objeto de la politica
2. procesos cubiertos
3. niveles de firma admitidos
4. criterios para uso de firma avanzada o cualificada
5. responsables internos
6. evidencias que deben conservarse

## 2. Ficha de proceso digital

### Campos orientativos

- nombre del proceso
- area responsable
- tipo de acto o documento
- nivel de riesgo
- herramienta de confianza utilizada
- evidencia generada
- plazo de conservacion

## 3. Matriz de decisiones

### Ejemplo de preguntas

- interviene una persona fisica o una entidad
- existe riesgo elevado de impugnacion
- se necesita prueba temporal reforzada
- hay comunicacion con relevancia formal
- hace falta reconocimiento transfronterizo

## 4. Registro de evidencias

### Campos orientativos

- identificador del expediente o transaccion
- fecha y hora
- tipo de evidencia
- servicio utilizado
- responsable de custodia
- estado de validacion

## 5. Checklist documental minima

### Elementos orientativos

- documento principal
- evidencia de firma o sello
- referencia temporal
- registro de envio o recepcion si procede
- trazabilidad de validacion

## Advertencia de uso

Estas plantillas no sustituyen:

- asesoramiento juridico
- analisis sectorial
- diseno tecnico especifico

Su utilidad es didactica y organizativa, para ayudar a pensar como traducir el marco eIDAS a procesos reales.

## Resumen rapido

Las organizaciones pueden beneficiarse de modelos internos sencillos que les ayuden a decidir que servicio usar, que evidencia conservar y como documentar sus procesos digitales.
