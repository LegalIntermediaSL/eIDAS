# 38. Tablas-resumen del tutorial

## Introduccion

Esta seccion sintetiza varias de las distinciones principales del tutorial en formato breve.

## 1. Firma, sello y certificado

| Elemento | Se vincula principalmente a | Funcion principal |
| --- | --- | --- |
| Firma electronica | Persona fisica | Manifestacion y firma de datos |
| Sello electronico | Persona juridica o entidad | Origen e integridad documental |
| Certificado | Persona, entidad o sistema | Vincular datos de verificacion |

## 2. Niveles de firma

| Nivel | Formalidad | Robustez probatoria orientativa |
| --- | --- | --- |
| Firma simple | Baja | Variable |
| Firma avanzada | Media o alta | Reforzada |
| Firma cualificada | Alta | Muy alta |

## 3. Prestadores

| Tipo | Puede prestar servicios cualificados | Supervision reforzada | Presencia en TSL |
| --- | --- | --- | --- |
| Prestador cualificado | Si | Si | Si |
| Prestador no cualificado | No | No en el mismo nivel | No en ese sentido |

## 4. Servicios relevantes

| Servicio | Funcion principal |
| --- | --- |
| Sello de tiempo | Acreditar momento temporal |
| Entrega electronica certificada | Acreditar envio y recepcion |
| Autenticacion de sitios web | Acreditar identidad del sitio |
| Conservacion y validacion | Mantener y verificar utilidad probatoria |

## 5. eIDAS y eIDAS 2.0

| Etapa | Idea central |
| --- | --- |
| eIDAS original | Identificacion electronica y servicios de confianza |
| eIDAS 2.0 | Wallet europea e identidad digital reutilizable |

## Resumen rapido

Las tablas-resumen ayudan a fijar las diferencias esenciales del marco eIDAS de forma rapida y visual.
